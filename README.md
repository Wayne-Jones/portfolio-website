# Portfolio Website

Personal portfolio website for Wayne Jones — Software Engineer & Photographer based in New York, NY.

## Stack

- **Framework**: React 19 + TypeScript
- **Build**: Vite 8 + TanStack Router (file-based routing)
- **Styling**: Tailwind CSS v4 + Base UI primitives
- **Fonts**: Self-hosted Inter Variable + Switzer (font-display: optional)
- **Lint/Format**: Oxlint + Oxfmt
- **Content**: Adapter pattern (Ghost CMS ready, mock fallback)

## Routes

| Route | Description |
|-------|-------------|
| `/` | Homepage — hero, work portfolio, photography, recent posts, contact |
| `/blog` | Blog index with pagination |
| `/blog/:slug` | Individual blog post |
| `/photography` | Photography gallery (masonry) |
| `/portfolio` | Work portfolio index |
| `/portfolio/:slug` | Individual project case study |

## Getting Started

```bash
# Install dependencies
pnpm install

# Development server
pnpm dev

# Production build
pnpm build

# Preview production build
pnpm preview

# Lint
pnpm lint

# Format
pnpm format
```

## Motion System

- **Scroll Reveal**: `[data-reveal]` elements fade/slide up on 15% visibility (IntersectionObserver)
- **Hero Entrance**: Staggered fade+slide (intro → headline → location → portrait)
- **Parallax**: Hero portrait parallax on scroll (desktop only, rAF)
- **Shimmer**: Skeleton loading animation (`.animate-shimmer`)
- **Reduced Motion**: All animations disabled via `@media (prefers-reduced-motion: reduce)`

## Fonts

Self-hosted with `font-display: optional` to avoid CLS:

- **Inter Variable** — UI text, body copy
- **Switzer Variable** — Display headings, hero text

Both served as WOFF2 with unicode-range subsetting.

## Content Adapter

The content layer uses an adapter pattern:

```typescript
// src/content/index.ts
export const contentService = createContentService()
```

- `GhostAdapter` — Production (requires Ghost CMS + API key)
- `MockAdapter` — Development/fallback (in-repo mock data)

Switch via `VITE_CMS_TYPE=ghost|mock` in `.env`.

## View Transitions

Cross-page view transitions via `@view-transition` on route elements, scoped to main content area. Root layout opts out via `view-transition-name: none`.

## Accessibility

- Semantic HTML, proper heading hierarchy
- Skip link, focus-visible rings
- `prefers-reduced-motion` respected globally
- Color contrast ≥ WCAG AA
- ARIA labels on icon-only controls

## Deployment

Static output to `dist/` — deploy anywhere (Vercel, Netlify, Cloudflare Pages, GitHub Pages, etc.).

```bash
pnpm build  # outputs to dist/
```
